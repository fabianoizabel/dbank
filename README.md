# dbank
Desafio Técnico, banco digital

# Para executar o projeto:
1 - Com o GIT instalado em sua máquina, clone o projeto através da URL https://github.com/fabianoizabel/dbank.git
2 - Instale o DOCKER em sua máquina.
3 - Após o DOCKER instalado, abra o terminal e vá até a pasta /diretorio-onde-o-projeto-foi-clonado/dbank-backend e execute o comando: docker compose -f docker-composer.yml up -d
4 - Após a instalação dos containers da aplicação com sucesso, importe o projeto no Spring Tool Suite
